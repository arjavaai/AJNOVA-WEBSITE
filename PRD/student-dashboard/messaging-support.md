# Student Dashboard - Messaging & Support

## Document Overview
This document details the Messaging & Support feature that enables communication between students and counsellors.

---

## 1. Purpose

The Messaging & Support module enables students to:
- Send messages to assigned counsellors
- Receive messages from counsellors
- Attach files to messages
- View message history
- Receive real-time notifications
- Get quick support responses

---

## 2. Messaging Interface

### 2.1 Inbox View
**Layout:**
- Left sidebar: Conversation list

**Message Thread:**
