# AJ NOVA - Design System

Based on the AJ NOVA Brand Kit.

## 1. Color Palette

### Primary Colors
| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **AjNova Navy** | `#0A2342` | Main brand color, text, header, footer, logo |
| **AjNova Coral** | `#F25C45` | Accent color, logo, icons, CTA buttons |
| **Warm White** | `#F8F8F6` | Main page background (Soft off-white) |

### Secondary Colors
| Color Name | Hex Code | Usage |
| :--- | :--- | :--- |
| **Slate Blue** | `#1B3A57` | Darker shade (footer, logo shadows) |
| **Soft Gray** | `#E2E2E2` | Section dividers |
| **Light Gray** | `#F1F1F1` | Light section backgrounds |
| **Peach Tint** | `#F3D6CB` | Decorative soft rounded shape accents |
| **White** | `#FFFFFF` | Card backgrounds, clean spaces |

## 2. Typography

### Primary Font: **Poppins**
*   **Usage:** Headings, Sub-headings, Titles.
*   **Weights:** SemiBold (600), Bold (700).

### Secondary Font: **Inter**
*   **Usage:** Paragraph text, body content, UI elements.
*   **Weights:** Regular (400), Medium (500).

### Optional Font: **Montserrat**
*   **Usage:** Alternative for specific design elements if needed.

## 3. UI Elements Styling

*   **Buttons:**
    *   **Primary:** Background `#F25C45` (Coral), Text `#FFFFFF` (White). Rounded corners.
    *   **Secondary:** Border `#0A2342` (Navy), Text `#0A2342` (Navy).
*   **Backgrounds:**
    *   Use `#F8F8F6` (Warm White) for the main page background to reduce eye strain compared to pure white.
    *   Use `#F1F1F1` (Light Gray) for alternating sections.
*   **Text:**
    *   Headings: `#0A2342` (Navy).
    *   Body: `#0A2342` (Navy) or a slightly lighter shade for readability.
