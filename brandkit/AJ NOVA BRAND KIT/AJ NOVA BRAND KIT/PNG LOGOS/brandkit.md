## **Typography**

| Type          | Font        | Usage                              |
| ------------- | ----------- | ---------------------------------- |
| **Primary**   | **Poppins** | Headings & Sub-headings            |
| **Secondary** | **Inter**   | Body text                          |
| **Optional**  | Montserrat  | Alternate heading or UI text style |



---

## **Usage Notes**

* Use **AJNova Navy** for large headlines and essential text.
* Use **Coral** sparingly for calls-to-action, highlights, and emphasis.
* Maintain **ample white/soft-gray space** for a clean, modern feel.
* Typographic hierarchy should follow:

  ```
  H1 → Poppins Bold
  H2 → Poppins Semi-Bold
  H3 → Poppins Medium
  Body → Inter Regular
  ```
